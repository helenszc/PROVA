1-o que é clone?
é uma cópia do repositório.

2 - O que é commit?
é o empacotamento das alterações realizadas no projeto.

3 - O que é um repositório?
é a pasta do projeto.

4 - O que é merge?
é a mesclagem de duas ou mais branches.

5 - O que é pull?
é puxar as informações do repositório remoto para o local .

7 - O que é branch?
são as ramificações do repositório.
